What you are looking at

Each bar spans the task start to end week. Bold tasks marked with a star are on the critical path — any delay pushes the end date. Milestones are zero-duration checkpoints. The red vertical line is today. Use the filter controls to view by phase, owner or status, and hover any bar for full task details.

What most PMs miss

Most PMs build a Gantt but forget to baseline it. Without a frozen baseline you cannot measure variance — you are simply editing history. Lock the baseline on kickoff day and treat every subsequent change as a formal change request with a date stamp. This single habit separates reactive PMs from proactive ones.

The expert tip

When a sponsor demands schedule compression, pull up the critical path filter first. Only tasks with zero float can shorten the end date. Crashing a non-critical task wastes money and looks incompetent in a steering committee. Show the critical path and point to exactly which tasks need resources — that turns the conversation from opinion into fact.
